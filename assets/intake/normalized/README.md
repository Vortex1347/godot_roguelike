# Normalized GLB

Эта зона предназначена только для чистых `GLB`-файлов, которые уже готовы к импорту в Godot.

## Что здесь должно быть

- правильный scale
- предсказуемая ориентация
- чистые mesh/material names
- без лишнего мусора сцены

## Что здесь не должно быть

- архивы
- сырые `FBX` / `Blend`
- финальные gameplay wrapper-сцены
