# Raw Asset Drop

Сюда можно просто складывать скачанные архивы или распакованные исходники без переименования и чистки.

## Рекомендованные подпапки

- `assets/intake/raw/environment_sanctum/`
- `assets/intake/raw/character_mage/`
- `assets/intake/raw/animations_humanoid/`
- `assets/intake/raw/focus_props/`
- `assets/intake/raw/magic_vfx/`

Если заливаешь ассеты вручную, просто клади их сюда по категориям, а дальше я уже нормализую их в `GLB` и подключу через proxy layer.
