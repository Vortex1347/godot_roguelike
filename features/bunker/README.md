# Bunker Feature

Зона для bunker hub, base utilities, trader access, quest handoff и будущих base upgrades.

## Текущий baseline

- Активный visual benchmark: `3D tilted-ortho bunker hall`.
- `bunker_slice_3d.tscn` является основным startup маршрутом.
- Старый `2D bunker` остается только как временный reference slice до отдельной cleanup-фазы.
- Все временные props для bunker, которые должны заменяться моделлером, должны идти через `assets/proxy/`.
